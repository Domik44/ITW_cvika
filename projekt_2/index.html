Airsoft Organizer

Description

In one of my school projects we had to come up with and idea for an application, which we would then implement (mainly focusing on frontend). I came up with an idea that me and 2 of my collegues could create and application where organizators could share information about their airsoft events and players could view those events and register to them if they wish to attend them. And so Airsoft Organizer was born. I mainly focused on setting up the database, creating backend background used for demonstrating how the frontend works and implementing frontend for events that are created and viewed by users of the application. I also added some functions that I would personally like as adding comments, possibility to review events, invite friends to events and more. My teammates then implemented frontend for arenas where the events are held at and teams, which users could create, add their friends to and present it to other users. The app also has very light and honestly not very working chat. I plan to work on this project more in the future and redo a lot of stuff.

GedHelp

Description

GedHelp is a web application that I created for my bachelors work. The goal of my thesis was to design and implement application which is capable of advising it's user, genealogist, suitable parish books for searching information about missing events of practicular person. The base of this application is processing input GEDCOOM file, creating records for missing events of people, choosing of suitable parish books and offering of these books to user through a graphical user interface, which is in our case is web browser. It was decided that the web application would be implemented in PHP using Laravel framework. Processing of GEDCOM files followed by mapping parish books to created records was implemented in Python. As part of this work I also processed all 8 digital archives containing parish books in Czech Republic. You can find more information about this project on my Github repo.

Airsoft

Airsoft is highly expanded sport in Czech Republic nowadays. It is a sport where people simulate military missions, using guns that shoot small pellets. It is considered as adrenaline sport and is good for letting the steam out of your head. It also helps with building good physical health as you move a lot, helps you with dealing with stressfull situations and quick thinking and also with orientation skill (I still get lost quite easily tho). Airsoft has been my passion for a long time, more than 6 years I would say. I play it with my friends, we have our own team. We play mainly in summer, usually on Sundays.
